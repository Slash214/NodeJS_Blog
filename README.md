# NodeJS开发博客系统
* 刚刚入门学习的Node，没有使用Koa2框架和express框架。开发的博客系统，完善了基本功能！

## 功能实现
* 博客列表的查询（keyword搜索）
* 文章列表的增加
* 文章列表的删除
* 文章列表的修改
* 登录用户的判断，是否为当前用户(密码加密。xss保护)
* 使用了Redis和Session存储用户信息
* 模板复用的封装

## 使用方法
- 下载先安装依赖： npm install
- npm run dev  运行
- 开发环境为 dev  线上环境为 prd
- mysql链接好像未使用连接池，如果服务器运行需要改变一下。
